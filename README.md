Yester Maps
======

Crunch today's osm-qa-tiles and count what happened yesterday, last week, and this month

Exported Properties: 

z15 Features:

```
{ type : "Feature"
  geometry: <Point @ Center of z15 tile >
  properties: {
  
    Y_uCount: #    
    W_uCount: #
    M_uCount: #
    
    bCountY: #
    bCountW: #
    bCountM: #
    
    kmCountY: #
    kmCountW: #
    kmCountM: #
    
    %Lorenz for buildings / roadkm
    
    New Features
    Edited Features
    
  }

```
